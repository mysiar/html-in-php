## Install

```
    composer require mysiar/html-in-php
```

## Tags implemented

* `<span>`
* `<div>`
* `<a>`
* `<p>`
* `<table>`
    * `<caption>`
    * `<tbody>`
    * `<thead>`
    * `<tfoot>`
    * `<tr>`
    * `<td>`
